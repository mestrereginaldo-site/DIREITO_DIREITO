# Site Advocacia Digital

Site profissional para advogado nômade digital, com sistema de agendamento e pagamentos online.

## Funcionalidades

- Design responsivo e profissional
- Sistema de agendamento em 4 etapas
- Apresentação de serviços jurídicos
- Formulário de contato
- Integração com WhatsApp
- Otimizado para conversão

## Como Publicar no Vercel

1. Faça commit de todos os arquivos no GitHub
2. Acesse [vercel.com](https://vercel.com)
3. Clique em "Add New" > "Project"
4. Importe o repositório do GitHub
5. Clique em "Deploy"

O site estará online em segundos!

## Personalização

1. Substitua as imagens pelas suas próprias
2. Atualize as informações de contato
3. Altere as cores no arquivo `style.css` (variáveis no topo)
4. Ajuste os preços dos serviços
5. Adicione seu número de OAB

## Integrações Futuras

Para tornar o site totalmente funcional, você pode integrar:

1. **Calendly** para agendamentos automáticos
2. **Stripe** ou **Mercado Pago** para pagamentos online
3. **EmailJS** ou **Formspree** para formulários
4. **Google Analytics** para métricas
